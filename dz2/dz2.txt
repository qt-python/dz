3. Ответьте на вопросы:
- Какой категории ui-элементов относится данный элемент? (см. доп. материалы)
	Это навигационные компоненты.

- Приведите пример - ui-элемента из категории Input Controls.
	Примером ui-элемента из категории Input Controls может быть текстовое поле, используемое для ввода текста пользователем.
	
- Является ли командная строка частью GUI?
	Командная строка не является частью графического пользовательского интерфейса (GUI). Она представляет собой текстовый интерфейс, 
	в котором пользователь вводит команды для выполнения определенных задач на компьютере. 
	GUI, напротив, предоставляет пользователю визуальные элементы, такие как кнопки, меню и окна, для выполнения действий на компьютере.



4. Какой тип мобильного приложения не имеет доступа к внутреннему функционалу смартфона, например пушам?

Тип мобильного приложения, который не имеет доступа к внутреннему функционалу смартфона, например пуш-уведомлениям, - это веб-приложения. 
	Веб-приложения запускаются веб-браузере на устройстве и они ограничены в 
	своих возможностях по сравнению с нативными приложениями, такими как доступ к уведомлениям пуш.
